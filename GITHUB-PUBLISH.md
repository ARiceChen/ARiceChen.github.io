# One-click GitHub Pages publishing

## First use

1. Double-click `publish-to-github.bat`.
2. If Git or GitHub CLI is missing, follow the installation prompt.
3. Sign in to GitHub in the browser when prompted.
4. The script creates a public repository named `YOUR_USERNAME.github.io`.
5. The site will be available at `https://YOUR_USERNAME.github.io`.

The first GitHub Pages deployment may take several minutes.

## Later updates

After editing the website, double-click `publish-to-github.bat` again. The
script commits and uploads the latest changes automatically.

## Important

- Do not place passwords or access tokens inside the website folder.
- GitHub Pages is public.
- Keep `index.html` in the website folder root.
