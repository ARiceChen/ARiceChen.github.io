# K. Chen — Academic Homepage

A concise single-page academic website for Kemi Chen.

The portrait is stored as `photo.jpg` beside `index.html`.

## Free GitHub Pages deployment

1. Create a public GitHub repository named `YOUR_USERNAME.github.io`.
2. Upload `index.html`, `style.css`, and `.nojekyll` to the repository root.
3. Open the repository's **Settings → Pages**.
4. Select **Deploy from a branch**, then choose `main` and `/ (root)`.
5. Visit `https://YOUR_USERNAME.github.io`.

## Updating the site

Edit `index.html` when adding publications, profile details, or new links.
